## Подсчёт слов в тексте (вариативная часть №2)

### Назначение программы
Программа анализирует текстовую строку, введённую пользователем, **удаляет знаки препинания**, подсчитывает частоту каждого слова и сохраняет результат в файл.

### Алгоритм работы

1. **Ввод строки**  
   Пользователь вводит произвольную строку текста с клавиатуры.

2. **Удаление знаков препинания**  
   Все знаки препинания (`. , ! ? ; : " ' ( ) [ ] { } < > @ # $ % ^ & * - + = | \ / ~`) заменяются на пробелы. Это позволяет правильно обрабатывать слова, после которых стоят запятые, точки и т.д.

3. **Нормализация регистра**  
   Все буквы переводятся в нижний регистр. Это нужно, чтобы варианты одного слова вроде "Cat", "cat" и "CAT" не считались разными.

4. **Разбиение на слова**  
   Строка разбивается на части по пробелам, табуляциям и переносам строк. Получается список отдельных слов.

5. **Подсчёт частоты**  
  Заводится пустой словарь. Каждое слово из списка обрабатывается так:
  - Если слово уже есть в словаре — увеличиваем его счётчик на 1.
  - Если встречается впервые — добавляем в словарь со значением 1.

6. **Вывод на экран**  
   Для каждого уникального слова программа выводит строку в формате:  
   `слово -> количество`

7. **Сохранение в файл**  
   Аналогичные пары «слово → количество» записываются в текстовый файл `result.txt` (кодировка UTF-8). Если файл с таким именем уже существует, он будет перезаписан.

### Пример работы

**Входные данные:** 

Введите строку: Hello, world! Hello, world. World is wonderful!


**Вывод на экран:** 

world -> 3
hello -> 2
is -> 1
wonderful -> 1

Результат сохранён в result.txt


**Содержимое файла `result.txt`:** 

world -> 3
hello -> 2
is -> 1
wonderful -> 1


### Тестирование программы

| Входная строка | Ожидаемый вывод на экран | Результат |
|----------------|--------------------------|-----------|
| `Hello, world!` | `hello -> 1`<br>`world -> 1` | ✔️ |
| `cat, cat. cat!` | `cat -> 3` | ✔️ |
| `one two-three four` | `one -> 1`<br>`two -> 1`<br>`three -> 1`<br>`four -> 1` | ✔️ |
| `Hello hello HELLO` | `hello -> 3` | ✔️ |
| `собака, кот кот!` | `собака -> 1`<br>`кот -> 2` | ✔️ |
| Пустая строка (просто Enter) | Нет слов для отображения, файл пустой | ✔️ |

### Особенности реализации

- Ключевое отличие от базовой версии – удаление пунктуации. Теперь "cat" и "cat," распознаются как одинаковое слово.
- Регистр не имеет значения – cat, Cat, CAT считаются одним словом.
- Разделители – слова отделяются друг от друга любыми пробельными символами (один пробел, несколько подряд, табуляция и т.д.).
- Файл с результатами – result.txt появляется в той же папке, где запущена программа.
- Дефисы и другие знаки внутри слов – тоже удаляются, поэтому "two-three" превращается в два отдельных слова: "two" и "three".

Вариативная часть №2 добавляет очистку от знаков препинания перед подсчётом, что делает анализ более точным. Полный процесс включает: получение строки, фильтрацию пунктуации, нормализацию регистра, разбиение на слова, заполнение словаря со счётчиками, вывод на экран и запись в файл.
